# NexusAI Playground

> Browser web + API tester OpenAI-compatible en un solo archivo HTML. Sin dependencias de servidor, cero instalación.

## Uso rápido

Abrí `nexusai-playground.html` directamente en tu navegador, o usá GitHub Pages.

## Funciones

- **Navegador con pestañas** — abrí múltiples sitios en tabs simultáneos
- **API Tester con streaming** — conectate a cualquier endpoint OpenAI-compatible (HuggingFace Spaces, Groq, OpenRouter, servidor local)
- **Accesos rápidos** — HuggingFace, OpenRouter, Groq, n8n, Colab, GitHub
- **Configuración persistente** — Base URL, API Key, modelo, system prompt, temperatura
- **Búsqueda integrada** — Google, DuckDuckGo, Bing, Perplexity, Kagi
- **Light/Dark mode** con toggle
- **Atajos de teclado**: `Ctrl+T` nueva pestaña, `Ctrl+W` cerrar, `Ctrl+L` URL bar, `Ctrl+Enter` enviar API

## Configuración del Space HuggingFace (Gemma 4 E4B)

1. Abrí la app y hacé click en ⚙️
2. **Base URL**: `https://tu-usuario-space.hf.space/v1`
3. **Modelo**: `gemma-4-e4b-it`
4. **API Key**: dejar vacío (si no tenés auth)
5. Guardá y usá el **API Tester** para probar
